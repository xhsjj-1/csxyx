# 🎮 简单跑酷游戏网站

一个完整的HTML5跑酷游戏网站，支持手机和电脑游玩。完全响应式设计，无需安装，直接在浏览器中运行。

## 🌐 在线演示

- **主游戏**: `index.html`
- **测试页面**: `test-overlay.html` (界面显示测试)
- **优化测试**: `test-view.html` (游戏画面优化)
- **改进报告**: `test-improvements.html` (完整改进报告)

## 🚀 快速开始

### **方法1: 本地运行**
```bash
# 使用Python启动本地服务器
python -m http.server 8000
# 访问: http://localhost:8000
```

### **方法2: 使用部署脚本**
```bash
# Linux/Mac
chmod +x deploy.sh
./deploy.sh

# Windows
双击 deploy.bat
```

## 🌍 部署为网站

### **免费托管平台**

#### **1. GitHub Pages（推荐新手）**
```bash
1. 创建GitHub仓库
2. 上传所有文件
3. 启用GitHub Pages
4. 访问: https://用户名.github.io/仓库名
```

#### **2. Netlify（简单快速）**
```bash
1. 注册Netlify账号
2. 拖拽文件夹到Netlify
3. 获得免费域名: *.netlify.app
```

#### **3. Vercel（适合开发者）**
```bash
1. 注册Vercel账号
2. 连接GitHub仓库
3. 自动部署: *.vercel.app
```

### **自定义域名**
1. 购买域名（如: yourgame.com）
2. 在托管平台添加自定义域名
3. 配置DNS记录
4. 等待生效（24-48小时）

### **文件说明**

- `game.js`: 游戏逻辑，包含错误处理

1. **修改游戏难度**: 编辑 `game.js` 中的 `updateDifficulty()` 函数
2. **修改颜色主题**: 编辑 `style.css` 中的颜色变量
3. **添加新功能**: 在 `game.js` 中添加新的游戏逻辑
